# 伝助改(仮称)

TRPGコミュニティ向けの、ログイン不要・URL共有型のスケジュール調整Webアプリ

---

## 特徴

* ログインなし(URLトークン方式)
* シンプルで軽量な設計
* 予定の再入力が不要

---

## 使い方(想定)

### 1. セッションの作成
* セッションを新規作成する
* セッション名や説明文、セッションの候補日を指定可能
* 作成時に、共有用URLトークンと編集用URLトークンを発行
* 編集用URLから、セッション名や説明文等を編集可能

### 2. セッションへの参加
* セッションへ予定を追加する
* セッション画面から、「新規予定」又は「既存予定」を追加できる
* 予定追加時には(そのセッション内での)表示名、(そのセッション内での)メモ、(そのセッション内で)IDを公開するか、を設定する
* 新規予定にはIDが発行され、別セッションで既存予定として追加できる

### 3. 予定の編集
* 予定の編集を行う
* セッション画面から、IDが既知の予定の編集画面を開く
* IDが非公開の予定の編集には、正しいIDの入力を求める
* カレンダーのような画面から1時間単位で、利用可能/不可能/未入力 の3値を保存できる
* 同IDの予定はセッション間で共有されるので、複数セッションに参加していても複数回の入力は不要
* そのセッションについての情報・設定(表示名等)も編集できる

### 4. スケジュールの調整
* セッション画面から、参加者全員の予定が閲覧できる
* 全員が参加可能な時間/不明な時間などを色分けしてカレンダーのように表示できる
